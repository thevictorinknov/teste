Introduction

This repository houses and provide the frontend componentes for ALESP. Inside ALESP we're using micro-frontends architeture.

Getting Started

To get started with the cdnComponents frontend, follow the instructions below.

Pre Requisites

Installation

  1. Import the following functions inside the "componentController.js" file in your desired branch:

    
function cdnProvider(componentController) {
   return componentsCDN + componentController;
}

export async function importComponent(componentController) {
   const component = await import(cdnProvider(componentController));
   return component.default;
}

  2. Ensure that in your desired branch, at the file "utils.js" has the following variables:

    const contextoCDN = "sgrh-dev-componentes";
const componentsCDN = contextPath + "/" + contextoCDN + "/components/controllers/";
const originCDN = contextPath + "/" + contextoCDN + "/components/";

Usage

Once you have the project set up, you can start using the CDN Components. At the begging of your js file, import the module:

import { importComponent } from 'path/to/componentController.js'

Then you can start importing components from our CDN using the following syntax:

let C166 = await importComponent('o-166-controller.js')
const MC166 = Object.assign(Object.create(C166));
await MC166.init({ visible: true });
window.MC166 = MC166;
